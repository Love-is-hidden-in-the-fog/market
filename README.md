本项目是一个基于微信开发者工具的微信小程序项目

主要页面对应的文件夹如下：

|    页面名称    |  对应文件夹  |
| :------------: | :----------: |
|      首页      |    index     |
|    分类页面    |   category   |
|  商品列表页面  |  goods_list  |
|  商品详情页面  | goods_detail |
|   购物车页面   |     cart     |
|    收藏页面    |   collect    |
|    订单页面    |    order     |
|    搜索页面    |    search    |
|  个人中心页面  |     user     |
|  意见反馈页面  |   feedback   |
|    登录页面    |    login     |
|    授权页面    |     auth     |
|    结算页面    |     pay      |
| 封装自定义组件 |  components  |
|    封装请求    |    reques    |
